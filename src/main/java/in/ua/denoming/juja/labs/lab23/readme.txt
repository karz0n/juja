Алгоритм реализует поворот трехмерной (объемной) матрицы.

Метод rotateClockwise класса MatrixUtils выполняет следующие действия:
1. Проверяет, что метод получил трехмерную матрицу (допустимые размеры 1x1x1, 2x2x2, 3x3x3, ...) иначе возвращает null;
   (возможные ошибки: аргумент равен null, "длина" / "ширина" / "высота" не равны, есть строки разной длины, есть строки с null вместо одномерных/двумерных массивов)
2. Поворачивает трехмерную матрицу относительно главной диагонали проходящей от элемента [0][0][0] до элемента [L-1][L-1][L-1] (L - длина "грани куба") массив по часовой стрелке на 120 градусов
   (по часовой стрелке - если смотреть вдоль оси от элемента [L-1][L-1][L-1] в сторону элемента [0][0][0])

[[[1]]] -> [[[1]]]

[  [[1, 2], [3, 4]], [[5, 6], [7, 8]] ] -> [ [[1, 5], [2, 6]], [[3, 7], [4, 8]] ]

[ [[10, 11, 12], [20, 21, 22], [30, 31, 32]], [[40, 41, 42], [50, 51, 52], [60, 61, 62]], [[70, 71, 72], [80, 81, 82], [90, 91, 92]] ] ->
[ [[10, 40, 70], [11, 41, 71], [12, 42, 72]], [[20, 50, 80], [21, 51, 81], [22, 52, 82]], [[30, 60, 90], [31, 61, 91], [32, 62, 92]] ] ...
